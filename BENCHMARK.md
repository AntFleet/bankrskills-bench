# AntFleet Benchmark — bankrskills

This repository is a public benchmark mirror used by [AntFleet](https://antfleet.dev) — a two-model PR review system that posts only findings both models agree on, with SHA-pinned receipts.

It is **not maintained** and **not affiliated** with BankrBot. The real project lives at [BankrBot/skills](https://github.com/BankrBot/skills).

All reviews appear publicly on [antfleet.dev/receipts](https://antfleet.dev/receipts), pinned to the upstream SHAs.

## How this works

1. Each benchmark target is a real upstream change on `BankrBot/skills`.
2. The change is replayed as a PR on this mirror.
3. AntFleet's GitHub App reviews the PR automatically: Anthropic `claude-opus-4-7` and OpenAI `gpt-5` run in parallel, and only unanimous findings post.
4. Receipts appear publicly on [antfleet.dev/receipts](https://antfleet.dev/receipts).

## Methodology

Each replay PR is generated by cherry-picking the upstream commit onto a branch from its parent, then opening a PR against this mirror's default branch. The resulting diff is identical to the upstream diff. AntFleet's review runs against that diff. Nothing is merged.

## Why benchmark this repo

`BankrBot/skills` is the canonical skills registry for Bankr — an AI agent on Base that equips builders with plug-and-play tools for on-chain interactions. New skill additions are security-critical: each skill executes on-chain transactions on behalf of users. AntFleet benchmarks new skill PRs to surface issues before they reach production agents.

## Disclaimer

No upstream source files are copied or modified. Every replayed PR cherry-picks an upstream commit and preserves original authorship metadata. This mirror exists for research and commentary purposes (PR review benchmarking). To stop being benchmarked: open an issue on [antfleet/antfleet](https://github.com/antfleet/antfleet) and we will delete this mirror.
